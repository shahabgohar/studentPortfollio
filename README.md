## Build Setup

```bash
this is still the basic version of my portfollio more changes are comming soon
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
